Este projeto foi desenvolvido para a disciplina de **Desenvolvimento em JavaScript**. O objetivo da atividade era criar uma animação simples utilizando a API Canvas do navegador, sem o uso de bibliotecas externas.

## Sobre o Projeto

Seguindo o roteiro da aula prática, criei uma página web onde um círculo centralizado muda de cor automaticamente. Para isso, utilizei JavaScript puro e o método `requestAnimationFrame`, garantindo que a animação rodasse de forma fluida e performática.

### Requisitos atendidos
- [x] Criação do arquivo HTML com a tag `<canvas>`.
- [x] Configuração do contexto 2D no JavaScript.
- [x] Desenho de objetos (Círculo) usando `ctx.arc`.
- [x] Loop de animação com `requestAnimationFrame`.
- [x] Mudança dinâmica de cores (usando HSL).
